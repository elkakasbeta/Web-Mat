Componentes en Angular

La base de toda aplicación moderna con estructura modular.

Data Binding

Comunicación entre la vista y la lógica del componente.

Directivas Básicas

*ngIf, *ngFor, ngClass y ngStyle: Control total del DOM.

¿Qué son los Componentes?

Estructura

Un componente contiene:

  • HTML (vista)
  • TS (lógica)
  • CSS (estilos)
Ciclo de vida
  • constructor()
  • ngOnInit()
  • ngOnDestroy()
Standalone

Los componentes standalone permiten usar imports directamente sin módulos.

Tipos de Data Binding

Interpolation

Se usa {{ '{{ nombreDeVariable }}' }} para mostrar datos.

Property Binding

Se usa [prop]="valor" para pasar valores.

Event Binding

Se usa (evento)="accion()"

Two-Way Binding

Se usa [(ngModel)] para bidireccionalidad.
